# 猪妈妈的生日惊喜
基于 Qt + C++ 开发的休闲解谜小游戏，《高级语言程序设计》课程设计作品。

## 游戏介绍
本作是以小猪佩奇为主题的治愈系短篇解谜游戏。玩家跟随剧情流程，完成解锁烤箱谜题、烤制蛋糕、摆放装饰等互动操作，一步步制作生日蛋糕，最终为猪妈妈送上生日惊喜。

## 功能特色
- 多场景剧情流程切换
- 交互式解谜玩法（密码锁、道具收集）
- 音视频播放（背景音乐、结局视频）
- 完整通关流程与结束结算页面
- 「关于游戏」说明弹窗

## 技术栈
- 开发语言：C++
- 开发框架：Qt 6
- 图形绘制：QGraphicsScene
- 多媒体：QMediaPlayer

## 运行说明
### 方式一：直接运行打包程序（推荐）
1. 下载 `release` 文件夹中的打包好的程序压缩包
2. 解压后双击 `PeppaPuzzleCake.exe` 即可直接运行，无需额外配置环境

### 方式二：使用源码编译运行
1. 使用 Qt Creator 打开项目根目录下的 `.pro` 文件
2. 配置 Qt 6 开发环境，构建并运行
3. 鼠标点击场景物品互动，跟随提示完成流程即可通关
