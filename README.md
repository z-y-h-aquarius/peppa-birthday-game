# 猪妈妈的生日惊喜
基于 Qt + C++ 开发的休闲解谜小游戏。

## 游戏介绍戏简介:
这是一款以小猪佩奇为主题的休闲解谜小游戏，玩家将跟随剧情指引，寻找道具、完成简单谜题，一步步完成蛋糕制作的全部流程,最终给猪妈妈准备一场温馨的生日惊喜。
游戏画风轻松可爱，童趣柔和，操作简单易懂，最终见证完整生日蛋糕完成，收获满满的温馨与治愈感。

## 技术栈
- 开发语言：C++
- 开发框架：Qt 6.8.3

## 运行说明
### 方式一：直接运行打包程序（推荐）
1. 夸克链接：https://pan.quark.cn/s/607060a5bec6  
   提取码：CMgE
2. 下载后解压压缩包，打开文件夹，双击里面的.exe程序即可运行

### 方式二：使用源码编译运行
1. 使用 Qt Creator 打开项目根目录下的 `.pro` 文件
2. 配置 Qt 6.8.3 开发环境，构建并运行
